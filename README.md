Right hand and left hand Motion Analysis
This project analyzes left-hand and right-hand motion using accelerometer and gyroscope data collected from a mobile phone holding selfie camera by using phyphox Android application.
It uses time-series analysis and visualization to compare motion patterns and study the difference between right hand and left hand.

Features

Mobile sensor data processing
Time-series visualization
paire plot 
important feature 
Left vs right hand comparison
Correlation analysis
